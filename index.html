eveapi.cc

On the surface, EVEAPI is just another public API tool. In reality, it's whatever other EVE players pay for it to be. Most players aren't aware that API data can be falsified, and there are numerous insidious applications for those who are. By doctoring API data as it is pulled, fake emails, monetary transactions, and location data can be displayed alongside real and verifiable API data to give it a sense of reliability.

Built With : HTML/CSS/JS, MeteorJS
